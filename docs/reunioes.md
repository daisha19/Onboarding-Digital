# Reuniões

Esta página centraliza registros de reuniões, decisões e encaminhamentos do projeto.

## Como Registrar

Cada reunião deve conter:

- Data.
- Participantes.
- Pauta.
- Decisões tomadas.
- Próximas ações.
- Responsáveis.

## Modelo

```markdown
## Reunião - DD/MM/AAAA

### Participantes
- Nome

### Pauta
- Item discutido

### Decisões
- Decisão tomada

### Próximas ações
| Ação | Responsável | Prazo |
|---|---|---|
| | | |
```
