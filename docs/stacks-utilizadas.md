# Stack Tecnológica

## Frontend

| Tecnologia | Finalidade |
|---|---|
| React | Construção da interface do sistema |
| Vite | Criação e execução do projeto frontend |
| TypeScript | Tipagem do código no frontend |
| Tailwind CSS | Estilização da interface |

## Backend

| Tecnologia | Finalidade |
|---|---|
| Python | Linguagem utilizada no backend |
| FastAPI | Criação da API do sistema |
| SQLAlchemy | Comunicação entre backend e banco de dados |

## Banco de Dados

| Tecnologia | Finalidade |
|---|---|
| PostgreSQL 15 | Armazenamento dos dados do sistema |

## Infraestrutura e DevOps

| Tecnologia | Finalidade |
|---|---|
| Docker | Containerização da aplicação |
| Docker Compose | Execução dos serviços em conjunto |
| Google Cloud Platform | Hospedagem e infraestrutura em nuvem |
| GitHub Actions | Automação de testes, build e deploy |
