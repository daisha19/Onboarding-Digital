#  Digital Onboarding System

Sistema web para onboarding digital de colaboradores, desenvolvido para departamentos de RH realizarem o envio, gerenciamento e auditoria de documentos admissionais de forma segura e centralizada.

---


#  Stack Utilizada

## Frontend
- React 18
- Vite
- TypeScript
- Tailwind CSS

## Backend
- FastAPI
- SQLAlchemy
- JWT Authentication
- PostgreSQL

## Infraestrutura
- Docker
- Google Cloud Run
- Google Cloud Storage
